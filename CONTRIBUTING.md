# 🤝 Contributing to EduChain

Thank you for considering contributing to EduChain! This project aims to make education accessible through blockchain technology.

## How to Contribute

### Reporting Bugs
- Use GitHub Issues
- Include browser, OS, and MetaMask version
- Provide steps to reproduce
- Include screenshots if applicable

### Suggesting Features
- Open a GitHub Issue with "Feature Request" label
- Describe the feature and its benefit
- Consider SDG alignment (SDG 4 & 10)

### Code Contributions

1. **Fork the repository**
2. **Create a feature branch:**
   ```bash
   git checkout -b feature/amazing-feature
   ```
3. **Make your changes**
4. **Test thoroughly**
5. **Commit with clear messages:**
   ```bash
   git commit -m "Add: Amazing new feature"
   ```
6. **Push to your fork:**
   ```bash
   git push origin feature/amazing-feature
   ```
7. **Open a Pull Request**

## Development Guidelines

### Smart Contracts
- Use Solidity 0.8.20+
- Follow OpenZeppelin standards
- Add comprehensive comments
- Include security checks (ReentrancyGuard, etc.)
- Write tests for new functions

### Frontend
- Use Next.js and React best practices
- Follow existing component structure
- Use Tailwind CSS for styling
- Add Framer Motion animations thoughtfully
- Ensure responsive design (mobile-first)

### Code Style
- Use ESLint configuration
- Format with Prettier
- Clear variable names
- DRY principle

## Pull Request Guidelines

### PR Title Format
- `Add: New feature description`
- `Fix: Bug description`
- `Update: Component/file description`
- `Refactor: What was refactored`

### PR Description Should Include
- What changes were made
- Why they were necessary
- How to test the changes
- Screenshots (if UI changes)
- Related issue number

## Testing Checklist

Before submitting PR:
- [ ] Code compiles without errors
- [ ] All existing tests pass
- [ ] New features have tests
- [ ] UI is responsive
- [ ] MetaMask integration works
- [ ] No console errors
- [ ] Documentation updated

## Community Guidelines

- Be respectful and inclusive
- Help others learn
- Focus on constructive feedback
- Remember: we're all here to improve education equity

## Questions?

Open a GitHub Discussion or reach out to maintainers.

Thank you for making education more accessible! 🎓
